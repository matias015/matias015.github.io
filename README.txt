Instrucciones para ver el sitio:

1. Abre el archivo "index.html" en tu navegador web favorito (Chrome, Edge, Firefox).
2. Navega a través de la página principal.
3. Haz clic en "Explorar Sala" en las tarjetas de "Gran Palacio Central" o "Neon Verse Club" para ver las páginas de detalle de cada sala.

Estructura de archivos:
- index.html: Página principal de la cadena.
- room-grand.html: Página específica de la sala "Gran Palacio Central".
- room-neon.html: Página específica de la sala "Neon Verse Club".
- style.css: Estilos globales.
- script.js: Lógica para animaciones y scroll.
- images/: Carpeta de imágenes.
