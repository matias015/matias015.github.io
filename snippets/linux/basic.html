Estructura de directorios

Estructura básica de directorios en Linux

/ — Es el directorio raíz (root). Todo el sistema de archivos parte de aquí.

Dentro de / hay muchos directorios importantes, por ejemplo:

  • /home — Directorios personales de los usuarios.
  • /etc — Archivos de configuración del sistema.
  • /bin — Programas ejecutables básicos.
  • /usr — Aplicaciones y utilidades del sistema.
  • /var — Archivos variables, como logs.
  • /tmp — Archivos temporales.
  • /root — Directorio personal del usuario root (administrador).

Comandos básicos para trabajar con directorios

  • pwd — Muestra el directorio actual (path working directory).
  • ls — Lista los archivos y carpetas dentro del directorio actual o uno especificado.
  • cd nombre_directorio — Cambia al directorio especificado.
  • cd .. — Sube un nivel al directorio padre.
  • mkdir nombre_directorio — Crea un nuevo directorio.
  • rmdir nombre_directorio — Borra un directorio vacío.

Esquema de particiones

Una partición es una división lógica de un disco físico. Cada partición puede contener un sistema de archivos, como ext4, xfs, btrfs, etc., y Linux puede montar esas particiones en el sistema de archivos.

Esquemas de particionado más comunes

  • MBR (Master Boot Record): Soporta hasta 4 particiones primarias (o 3 primarias + 1 extendida con varias lógicas). Límite de 2 TB por disco.
  • GPT (GUID Partition Table): Reemplazo moderno de MBR. Soporta hasta 128 particiones y discos de varios terabytes (mucho más de 2 TB).

Comandos basicos

EL comando ls

muestra el contenido pero listado en formato lista ademas de mostrar permisos, número de enlaces, propietario, grupo, tamaño, fecha de modificación y nombre del archivo. 

    $ ls -l /home/mydir

El flag -a muestra archivos ocultos, que comienzan con un punto.

    $ ls -a /home/mydir
    $ ls -la /home/mydir

Ordenar por fecha (más reciente al final)

    $ ls -lt

Ordernar por tamaño

    $ ls -lS

Mostrar en formato "legible" por humanos (tamaños en KB, MB...)

$ ls -lh

Para ver todos los archivos y subdirectorios recursivamente:

    $ ls -R

Podemos mostrar el peso de archivos en mb con

    $ ls -l --block-size=M /...
