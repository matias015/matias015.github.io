EL comando ls

muestra el contenido pero listado en formato lista ademas de mostrar permisos, número de enlaces, propietario, grupo, tamaño, fecha de modificación y nombre del archivo. 

    $ ls -l /home/mydir

El flag -a muestra archivos ocultos, que comienzan con un punto.

    $ ls -a /home/mydir
    $ ls -la /home/mydir

Ordenar por fecha (más reciente al final)

    $ ls -lt

Ordernar por tamaño

    $ ls -lS

Mostrar en formato "legible" por humanos (tamaños en KB, MB...)

$ ls -lh

Para ver todos los archivos y subdirectorios recursivamente:

    $ ls -R

Podemos mostrar el peso de archivos en mb con

    $ ls -l --block-size=M /...
