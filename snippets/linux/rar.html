Archivos rar

Instalar rar y unrar

    sudo apt update
    sudo apt install unrar rar

    sudo yum install unrar rar

En algunos sistemas, rar puede no estar disponible libremente debido a licencias, pero unrar (para descomprimir) sí suele estar.

Descomprimir archivos Rar

Ver el contenido del archivo sin extraer

unrar l archivo.rar

Extraer en el directorio actual

unrar x archivo.rar

Extraer en un directorio específico

unrar x archivo.rar /ruta/destino/

Usa x para mantener la estructura de carpetas.

Comprimir a rar

rar a archivo.rar archivo1 archivo2

a: Añade archivos al archivo .rar

Se pueden comprimir carpetas también

rar a archivo.rar carpeta/

Utiles

  • unrar l Lista contenido del archivo .rar
  • unrar x Extrae con estructura de carpetas
  • unrar e Extrae todos los archivos en un solo nivel
  • rar a Comprime archivos
  • -p[contraseña] Para archivos .rar protegidos con clave
