Archivos zip

Zip es uno de los formatos de archivo más populares que existen. Con zip, puedes comprimir varios archivos en uno solo. Esto no sólo ahorra espacio en el disco, sino que también ahorra ancho de banda en la red. Por eso te encontrarás con archivos zip casi todo el tiempo. 

    sudo apt install zip unzip
    zip --version

Comprimir archivos con zip

Comprimir uno o más archivos 

    zip archivo.zip archivo1.txt archivo2.txt

Comprimir un directorio completo 

zip -r archivo.zip carpeta/

Comprimir excluyendo archivos específicos

    zip -r archivo.zip carpeta/ -x "*.log" "*.tmp"

Descomprimir archivos con unzip

unzip archivo.zip

Extraer en un directorio específico

    unzip archivo.zip -d destino/

Ver el contenido del archivo ZIP sin extraer

    unzip -l archivo.zip

Extraer solo ciertos archivos 

unzip archivo.zip archivo1.txt carpeta/archivo2.txt

Opciones utiles

Sobrescribir sin preguntar

    unzip -o archivo.zip

No sobrescribir archivos existentes

unzip -n archivo.zip

Preguntar antes de sobrescribir

    unzip -u archivo.zip
